# Практические работы 13–17 | Фронтенд и бэкенд разработка

## 13-14 — Service Worker + Web App Manifest

Офлайн-приложение для заметок на основе PWA.

- Заметки сохраняются в `localStorage`
- Service Worker кэширует статические файлы — приложение работает без интернета
- `manifest.json` с иконками позволяет установить приложение на устройство

**Запуск:** `npx serve . -p 3000` → `http://localhost:3000`

---

## 15-16 — App Shell + WebSocket + Push-уведомления

Расширение приложения заметок с архитектурой App Shell и real-time функциями.

- App Shell: навигация и каркас кэшируются отдельно от динамического контента (Network First для `/content/`)
- Socket.IO: при добавлении заметки все открытые вкладки получают уведомление в реальном времени
- Web Push: подписка/отписка на системные push-уведомления через VAPID

**Запуск:** `node server.js` → `http://localhost:3001`

---

## 17 — Детализация Push (напоминания)

Расширение практики 16 с функционалом напоминаний.

- Форма для создания заметки с выбором даты и времени напоминания
- Сервер планирует push-уведомление через `setTimeout` и хранит активные таймеры в `Map`
- Уведомление содержит кнопку «Отложить на 5 минут» — Service Worker перехватывает действие и отправляет запрос на сервер (`POST /snooze`)
- Напоминания срабатывают даже при закрытом приложении

**Запуск:** `node server.js` → `http://localhost:3001`
